# Simple Web3 Wallet Checker

This is a beginner-friendly Web3 project.

This project helps you:
- Connect a crypto wallet (MetaMask)
- Read wallet address
- Check which blockchain network you are on
- Get the native token balance (ETH, BNB, MATIC, etc.)

This project uses:
- HTML
- JavaScript
- ethers.js

## Features
- One-click wallet connect
- Show wallet address on screen
- Show network name
- Show wallet balance
- Clean and simple code

## Requirements
- MetaMask installed in browser
- Internet connection
- Basic knowledge of HTML and JavaScript

## How to Run
1. Download this repository
2. Open `index.html` in your browser
3. Click "Connect Wallet"
4. Allow MetaMask permission
5. See your wallet info on screen

## File Structure
- index.html -> Main webpage
- app.js -> Wallet logic
- wallet.js -> Blockchain functions
- config.js -> Network settings
- provider.js -> Web3 provider setup
- balance.js -> Balance checker
- network.js -> Network reader
- connect.js -> Wallet connect logic
- ui.js -> Screen updates
- utils.js -> Helper functions

## Learning Goal
This project is made to help beginners understand:
- How Web3 connects to blockchain
- How wallets talk to websites
- How to read blockchain data
